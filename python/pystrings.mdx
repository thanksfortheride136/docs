## Definition: 
A _string_ is a _data type_ that is a sequence of characterters(often referred to as chars), strings are used to hold text data. Stirngs can be enclosed in either single `''` or double `""` quotes. 

## Syntax:
```python
"string"
```
## Examples:
<CodeGroup>
```Python strings1.py
print("Strings are located inside of many different functions, methods")
```
</CodeGroup>

<Warning>A `String` is its own data type, in Python there are many data types. When performing certain operations you may run into a `TypeError`. For instance if you try adding a `String` and `int` data type, you would recieve an error.</Warning>
