<CardGroup cols={1}>
  <Card title="Python for Linux" icon="linux">
    1.Your linux distribution may come with linux, please note that install directions may vary based on your linux distro. Check if python is installed with: <br />`python3 --version`<br /><br />
    2.Open a new terminal with: <br /> `ctrl + alt + T`<br /><br />
    3. Use the following commands to install Python: <br />
    `sudo apt update`<br />
    `sudo apt install python3`
  </Card>
  <Card title="Python for Windows" icon="windows">
    1.Navigate to the Python downloads [**here**](https://www.python.org/downloads/) <br /><br />
    2.Download and open it on your computer. <br /><br />
    3. Follow windows system install instructions.
  </Card>
  <Card title="Python for MacOS" icon="apple">
    1.Navigate to the Python downloads [**here**](https://www.python.org/downloads/macos/) <br /><br />
    2.Download and open it on your computer. <br /><br />
    3. Follow MacOS system install instructions.
