## Definition: 
`left()` rotates the turtle oject left or right a set distance. You must use the name of your turtle object, the left()/right() method and then an integer as a parameter that represents the angle of rotation.

## Syntax:
```python
turtle_name.left(int or float)
```
```python
turtle_name.right(int or float)
```
## Examples:
<CodeGroup>
```python left.py
tt.left(90)
```
